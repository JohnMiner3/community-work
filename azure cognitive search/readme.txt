The purpose of this presentation is to show how to deploy services and setup azure 
cognative search to use the following services:  azure data lake storage, azure
sql server and azure cosmos db.

Here is a desciption of the directories.

cli - use azure cli to deploy services via cloud shell.
csv - 2017 stock data for s&p 500 companies
json - sample data for first stock.  total data is over 50 mb.
txt - the text for the grimm tales
various - text files with info for demo
