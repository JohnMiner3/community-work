Topic:

Running SQL Server on Linux


Abstract:

SQL Server running on Linux is one of the biggest changes to happen in some time at Microsoft.  The release of the 2017 version of the database engine is about to happen. 

As a database administrator, are you ready to support this version of the database in your environment?

This talk will focus on key concepts such as database migration, monitoring your environment, and providing high availability to your applications.  If you are going to support a SQL Server Linux environment in the near future, this is a must attend presentation.


Coverage:

1 - Migrating databases from Windows to Linux.
2 - Monitoring your environment with home grown and/or packaged software.
3 - Providing high availability to your applications.
4 - What is missing from the SQL Server stack?
